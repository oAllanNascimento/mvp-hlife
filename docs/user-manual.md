## Introdução

Bem-vindo ao Sistema de Agendamento HealthLife! Este manual foi desenvolvido para auxiliar todos os usuários a utilizarem o sistema de forma eficiente e produtiva.

O HealthLife é um sistema completo de agendamento projetado para simplificar o processo de marcação de consultas e gerenciamento de horários, permitindo que profissionais gerenciem sua disponibilidade e que clientes realizem agendamentos de forma autônoma.

## Visão Geral do Sistema

O sistema HealthLife oferece funcionalidades específicas para diferentes tipos de usuários:

- **Administradores**: Criam agendas e as vinculam a gestores específicos
- **Gestores**: Visualizam e gerenciam agendamentos vinculados a eles
- **Profissionais**: Visualizam agendamentos e marcam presença/ausência
- **Clientes Finais**: Realizam agendamentos via URL pública (sem necessidade de login)

## Para Administradores

### Acessando o Sistema

1. Acesse a página inicial do HealthLife através do seu navegador
2. Insira suas credenciais de administrador (nome de usuário e senha)
3. Clique no botão "Entrar"

### Criação de Agendas

Para criar uma nova agenda:

1. No painel de controle, clique no botão "Criar Agenda"
2. Preencha as informações básicas da agenda:
   - Nome da agenda
   - Descrição (opcional)
   - Tipo de serviço
   - Duração padrão dos agendamentos
3. Configure os horários disponíveis:
   - Dias da semana disponíveis
   - Horários de início e término
   - Intervalos entre agendamentos
4. Vincule a agenda a um gestor específico:
   - Selecione o gestor na lista disponível
   - Opcionalmente, adicione profissionais vinculados
5. Clique em "Salvar" para criar a agenda

### Gerenciando Agendas

Para gerenciar agendas existentes:

1. No painel de controle, acesse a seção "Agendas"
2. Visualize todas as agendas criadas em formato de lista
3. Utilize os filtros para encontrar agendas específicas
4. Clique em uma agenda para:
   - Editar suas configurações
   - Desativar temporariamente
   - Excluir permanentemente
   - Visualizar estatísticas de uso

### Gerenciando Usuários

Como administrador, você pode gerenciar todos os usuários do sistema:

1. Acesse a seção "Usuários" no painel de controle
2. Adicione novos usuários clicando em "Adicionar Usuário"
3. Defina o tipo de usuário (Administrador, Gestor ou Profissional)
4. Configure as permissões específicas
5. Gerencie usuários existentes (editar, desativar, excluir)

## Para Gestores

### Acessando o Sistema

1. Acesse a página inicial do HealthLife através do seu navegador
2. Insira suas credenciais de gestor (nome de usuário e senha)
3. Clique no botão "Entrar"

### Visualizando o Dashboard

O dashboard do gestor apresenta uma visão geral dos agendamentos:

1. Total de agendamentos
2. Agendamentos do dia
3. Taxa de comparecimento
4. Gráficos de desempenho

### Gerenciando Agendamentos

Para visualizar e gerenciar agendamentos:

1. Acesse a seção "Agendamentos" no menu lateral
2. Escolha entre visualização em calendário ou lista
3. Utilize os filtros para refinar a visualização:
   - Filtro por data (dia, semana, mês)
   - Filtro por status (confirmado, pendente, cancelado)
   - Filtro por profissional (se aplicável)
4. Clique em um agendamento específico para:
   - Ver detalhes completos
   - Editar informações
   - Cancelar o agendamento
   - Enviar lembrete ao cliente

### Gerando Relatórios

Para gerar relatórios de agendamentos:

1. Acesse a seção "Relatórios" no menu lateral
2. Selecione o tipo de relatório desejado
3. Configure os filtros específicos (período, profissionais, etc.)
4. Clique em "Gerar Relatório"
5. Visualize os dados na tela ou exporte para CSV/PDF

### Acompanhando a Performance

Para acompanhar a performance dos agendamentos:

1. Acesse a seção "Performance" no menu lateral
2. Visualize métricas como:
   - Taxa de ocupação
   - Taxa de cancelamentos
   - Taxa de comparecimento
   - Horários mais procurados

## Para Profissionais

### Acessando o Sistema

1. Acesse a página inicial do HealthLife através do seu navegador
2. Insira suas credenciais de profissional (nome de usuário e senha)
3. Clique no botão "Entrar"

### Visualizando Agendamentos

Para visualizar seus agendamentos:

1. No dashboard inicial, você verá os agendamentos do dia atual
2. Para uma visão mais detalhada, acesse a seção "Meus Agendamentos"
3. Utilize os filtros para refinar a visualização:
   - Filtro por data
   - Filtro por status
4. Clique em um agendamento para ver os detalhes completos

### Registrando Presença/Ausência

Para registrar a presença ou ausência de um cliente:

1. Localize o agendamento na lista
2. Clique no botão "Registrar Atendimento"
3. Selecione o status:
   - Cliente Compareceu
   - Cliente Não Compareceu
4. Adicione observações se necessário
5. Clique em "Salvar"

### Consultando Disponibilidade

Para consultar sua disponibilidade:

1. Acesse a seção "Minha Agenda" no menu lateral
2. Visualize os horários disponíveis e ocupados
3. Utilize os filtros de data para navegação
4. Bloqueie horários específicos se necessário:
   - Clique no horário desejado
   - Selecione "Bloquear Horário"
   - Defina o motivo e duração do bloqueio
   - Salve as alterações

## Para Clientes Finais

### Acessando a Página de Agendamento

1. Acesse a URL pública fornecida para a agenda específica
2. Você será direcionado para a página de agendamento público

### Realizando um Agendamento

Para realizar um novo agendamento:

1. Na página de agendamento, visualize as informações do serviço
2. Selecione uma data disponível no calendário
3. Escolha um horário entre as opções disponíveis
4. Preencha o formulário com suas informações:
   - Nome completo
   - Email
   - Telefone
   - Observações (opcional)
5. Confirme seus dados e clique em "Agendar"
6. Você receberá uma mensagem de confirmação na tela
7. Um email de confirmação será enviado para o endereço fornecido

### Cancelando um Agendamento

Para cancelar um agendamento existente:

1. Acesse o link de cancelamento disponível no email de confirmação
2. Confirme sua intenção de cancelar o agendamento
3. Você receberá uma confirmação do cancelamento
4. Um email de cancelamento será enviado para seu endereço

## Soluções de Problemas Comuns

### Problemas de Login

Se você estiver tendo problemas para acessar o sistema:

1. Verifique se está inserindo as credenciais corretas
2. Utilize a opção "Esqueci minha senha" se necessário
3. Limpe o cache do navegador e tente novamente
4. Se o problema persistir, entre em contato com o administrador do sistema

### Problemas com Agendamentos

Se você encontrar problemas ao criar ou gerenciar agendamentos:

1. Verifique se todos os campos obrigatórios foram preenchidos
2. Certifique-se de que o horário selecionado está disponível
3. Atualize a página para garantir que está vendo as informações mais recentes
4. Se o problema persistir, entre em contato com o suporte

### Problemas com Emails de Confirmação

Se não receber os emails de confirmação:

1. Verifique a pasta de spam/lixo eletrônico
2. Certifique-se de que o endereço de email foi inserido corretamente
3. Adicione o domínio do sistema à sua lista de remetentes confiáveis
4. Se o problema persistir, utilize o contato alternativo fornecido
